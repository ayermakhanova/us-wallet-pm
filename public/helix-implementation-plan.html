Q2 Core Banking Implementation DDA, Savings, ACH

Scope: Full core banking setup including account creation, sandbox build, Socure KYC certification, TAB Bank requirements for each phase (Alpha, Beta, GA), production environment build, and go-live. Sandbox testing runs throughout the entire implementation.

Debit Card Implementation CPI + Visa DPS

Note: Timeframes will vary based on established BOR (Bank of Record) relationship with VDPS. CPI custom card and carrier order can take up to 16 weeks - this is the longest lead-time item in the entire program.

Ensenta mRDC Implementation Mobile Remote Deposit Capture

Scope: Full mRDC integration including Ensenta certification (6-8 weeks client integration phase), SFTP setup, UAT environment creation, end-to-end testing, and limited production rollout before GA.

Mobile Wallet - Manual Provisioning Apple Pay, Google Pay, Samsung Pay

Prerequisite: VDPS BIN installation project must be completed. Bank agreements (Apple, Samsung, Google) must be in place. Manual provisioning must be completed before In-App provisioning can begin.

Mobile Wallet - In-App Provisioning Visa DPS SDK Integration

Prerequisite: Client must have completed the manual provisioning project. Visa Executive Sponsor Bank Agreement Form required (SLA ~15 days). Q2 sandbox connects to Visa DPS sandbox with hardcoded test cards.

Digital Issuance - Single PAN Instant Card Issuance

Scope: Enables instant digital card issuance where physical and digital card orders are initiated simultaneously with a single PAN. Review Helix API docs at https://sandbox-api.Helix.io/card/createDigital. The "Is digital only" flag should never be set to true.

Falcon Fraud Text Alerts Visa DPS / Helix

Scope: Enables real-time fraud text alert notifications for cardholders. Requires coordination between Helix, Visa DPS, and Ria for configuration, testing, and enablement.
