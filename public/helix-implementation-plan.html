Core Banking Implementation DDA, Savings, ACH

Timeline: Months 1-8 (Mar - Oct 2026). Sandbox build runs 8-12 weeks in parallel with requirements gathering. Production environment takes 7-10 days to build. Alpha testing is 6 weeks, Beta is 6 weeks.

Debit Card Program CPI + Visa DPS

Critical: CPI custom card and carrier order takes up to 16 weeks. This is the longest lead-time item. Card design proofs must be approved ASAP to start the clock.

Ensenta mRDC Mobile Remote Deposit Capture

Timeline: Client integration phase is 6-8 weeks. UAT environment takes 10+ business days after certification. Limited production testing is 3 weeks before GA.

Mobile Wallets Apple Pay, Google Pay, Samsung Pay

Dependency: Manual provisioning must be completed before In-App provisioning can begin. VDPS BIN installation must be done first. Apple approval SLA is ~1 week. Visa Executive Sponsor Agreement SLA is ~15 days.
